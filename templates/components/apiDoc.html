Introduction

This project implements a real-time face detection API using the state-of-the-art YOLOv5 object detection model. The API allows uploading images and returns precise bounding boxes around detected faces along with confidence scores. Key features include:

  • Fast, accurate face detection powered by the YOLOv5 model architecture
  • Detects faces at various scales and orientations, including small faces in crowded scenes
  • Returns bounding box coordinates and facial key points for each detected face
  • Simple REST API interface for easy integration into web and mobile applications

Getting Started

System Requirements:

  • Python 3.7+
  • PyTorch 1.7+
  • FastAPI
  • Docker (optional)

Installation

  1. Clone the repository: git clone https://github.com/username/yolov5-face-api.git
  2. Install dependencies: pip install -r requirements.txt
  3. Download the pre-trained YOLOv5 face detection model and place in models/

Running the API

  • Run locally: uvicorn main:app
  • Run in Docker: docker build -t yolov5-face-api . && docker run -p 8000:8000 yolov5-face-api
  • Access API docs at http://localhost:8000/docs

API Reference

  • POST /image/classify - Submit an image for face detection. Returns bounding boxes and confidence scores.

See the full API specification for request/response formats and example code.
