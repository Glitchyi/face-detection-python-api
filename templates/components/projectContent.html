Introduction

The Masuku API is a cutting-edge solution that leverages the power of the YOLO V5 object detection model to detect human faces and determine the presence of face coverings. This innovative API is designed to assist organizations in enforcing face covering policies and ensuring compliance in various settings where such measures are mandated for safety and public health reasons.

Working

The Masuku API provides a user-friendly interface that allows seamless integration with various applications and systems. It accepts image inputs via HTTP POST requests and supports commonly used image formats such as JPEG and PNG. Utilizing the state-of-the-art YOLO V5 model, the API analyzes the provided images to detect human faces and identify whether they are covered, partially covered, or uncovered by masks or other face coverings. Upon processing the input, the Masuku API generates a structured JSON response, indicating the face covering status along with associated confidence scores or probabilities. This valuable information can be seamlessly integrated into existing systems or applications, enabling real-time monitoring and enforcement of face covering policies.

Getting Started

System Requirements:

  • Supports CPUs on Windows 8.1+, Linux, and macOS with acceleration libraries like MLAS, MKL-DNN, MKL-ML
  • For NVIDIA GPUs, supports CUDA 10.0, cuDNN 7.3 (some Linux Python needs CUDA 10.1, cuDNN 7.6), Kepler and newer GPUs
  • Has execution provider for AMD GPUs with ROCm
  • On Windows, DirectML needs DirectX 12 capable GPUs like NVIDIA Kepler+, AMD GCN 1st Gen+, Intel Haswell+ integrated
  • Has providers for Intel OpenVINO, ARM Compute Library, Rockchip NPU, Xilinx Vitis-AI, Huawei CANN, Qualcomm QNN

Installation

  • Clone the repository: git clone https://github.com/masukuapi/masuku-docs
  • Install dependencies: pip install -r requirements.txt
  • Run the main python file: python main.py

Running the API (using Docker)

  • Clone the repository: git clone https://github.com/masukuapi/masuku-docs
  • Build the docker image: docker build . -t masuku-api:latest
  • Run the docker container: docker run -p 8000:8000 masuku-api
